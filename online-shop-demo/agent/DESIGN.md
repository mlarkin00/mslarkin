# Goal
Create a web app that can be used to simulate failure modes in the online shop demo.

# Tech
* Use Google ADK for Go
* Deploy agent to Vertex AI Angent Engine

# UI
* Simple web app with a form to select: Target Project (text, default "mslarkin-ext"), Target Cluster (drop-down populated by list_clusters), Failure Mode(s) (toggle buttons, default off, enable/disable each failure mode).
